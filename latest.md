**These documentation pages cover all functionality available in the Rancher Observability Self-hosted product**. 

Rancher Observability Self-hosted can be used to observe an extensive set of technologies that originate either in your own data centers or in the cloud. If you are focussed solely on cloud technologies, such as Kubernetes or AWS, you might consider our SaaS solution. 

➡️ Switch to the [documentation for Rancher Observability SaaS](https://docs.stackstate.com/v/stackstate-saas/)

## Supported Rancher Observability versions

The supported versions of Rancher Observability are listed below:

* **Rancher Observability v5.1** - You are currently reading the documentation for Rancher Observability version 5.1.
* **Rancher Observability v5.0** - [Switch to the v5.0 docs](https://docs.stackstate.com/v/5.0/).
* **Rancher Observability v4.6** - [Switch to the v4.6 docs](https://docs.stackstate.com/v/4.6/).

For details on changes in each release and how to upgrade Rancher Observability, check the [Rancher Observability release notes and upgrade instructions](setup/upgrade-stackstate/).

To find documentation for previous Rancher Observability versions, use the drop-down menu at the top left of this page.
